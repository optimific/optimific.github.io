Legislator Data

State legislator data sources:

  • Open States Data
  • State Legislature Websites

    • When possible data was retrieved using the Open States Data API in R. Otherwise, data was retrieved from state legislature websites and other publicly available information.

    Electoral Result Data

    Election result sources - State Legislative Elections:

  • Princeton Election Consortium State Legislative Elections, 1971-2016 data
  • Ballotpedia
  • State election websites

    • Election result sources - Presidential and other Election Results:

  • Daily Kos Elections', statewide election results by congressional and legislative districts
  • State election websites

    • Available data varied by state - some states had the results for every available election, while for others, no data was available for even presidential elections

    Demographic Data

    Demographic data was retrieved from 2016 5-year American Community Survey (ACS) estimates using Kyle Walker's fantastic R package Tidycensus
